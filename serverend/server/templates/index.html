Introduction
Nucleus Segmentation

Automatic segmentation of microscope images is an important task in medical image processing and analysis. Nucleus segmentation is a very important example of this task, which can speed up the research of different diseases. We tried to use 3 kinds of framework to sovle this problem

  1. CNN backbone + Transpose Convolution head.
  2. Fully Convolution Network(U-net)
  3. CNN backbone + Active Contour Adjustment(DALS)
based on 670 nucleus images dataset from Kaggle 2018 Datascience Bowl
